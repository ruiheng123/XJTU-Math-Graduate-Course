# 凸分析与优化理论

课程学时：12周 * 4 节 = 48 h

授课教师：李辉

考核方式：10% 考勤 + 20% 期末报告 + 70% 期末考试

---

## 讲义大纲

### 凸集基础理论

1. 凸集、仿射集、锥、相对内部
2. 超平面与超平面分离
3. 凸函数相关
4. 回收锥、极点和极锥
5. 多面体问题（非重点）

### 对偶理论

1. 共轭函数
2. Min-Max 理论、MCMC框架
3. 次微分与次梯度
4. 切锥与法锥

### 凸优化相关算法

1. 次梯度下降法
2. 临近点法
3. 梯度下降法
4. 近似点梯度法
5. ADMM 算法、加速算法
6. 随机梯度下降法、障碍函数法（非重点）
